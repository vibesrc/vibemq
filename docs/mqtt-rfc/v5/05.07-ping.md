## 5.7.1 PINGREQ – PING Request

The PINGREQ packet is sent from a Client to the Server to:
1. Indicate the Client is alive (Keep Alive)
2. Request the Server to respond to confirm it is alive
3. Exercise the network to indicate the connection is active

### Fixed Header

```
0xC0 0x00  (PINGREQ, Remaining Length = 0)
```

### Variable Header: None
### Payload: None

---

## 5.7.2 PINGRESP – PING Response

A PINGRESP is sent by the Server in response to a PINGREQ.

### Fixed Header

```
0xD0 0x00  (PINGRESP, Remaining Length = 0)
```

### Variable Header: None
### Payload: None

**[MQTT-3.12.4-1]** The Server MUST send a PINGRESP in response to a PINGREQ.

---

## 5.7.3 Usage

These are the smallest MQTT packets (2 bytes each).

```
Client                              Server
   │                                   │
   │  PINGREQ (0xC0 0x00) ──────────► │
   │                                   │
   │  ◄─────────── PINGRESP (0xD0 0x00)│
   │                                   │
```
